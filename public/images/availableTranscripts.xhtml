Customer File Number Help

The 10-digit Customer File Number serves as an alternative tracking number to match the transcript to the taxpayer for third-parties such as tax practitioners, lenders and college financial aid offices.

Leave the entry blank if you were not directed by a third-party to enter the number. Do not use your Social Security number.

Using your keyboard, you can manually enter a Customer File Number assigned to you, for example, by a lender or college financial office. That customer file number will display on the transcript when it is downloaded.
